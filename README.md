CreditWise Loan System 🏦
Project Overview
Built an end-to-end supervised ML pipeline using KNN, Logistic Regression, and Naive Bayes to predict loan approval.

Key Implementation Details:
Supervised Learning: Implemented an end-to-end pipeline for loan prediction.

Algorithms: Used KNN, Logistic Regression, and Naive Bayes models.

Classification: Performed Binary classification to determine approval status.

Data Science Workflow: Included EDA (Exploratory Data Analysis) and Feature Engineering.

Model Evaluation: Analyzed performance using metrics like Precision, Recall, and F1-Score.
